# Vehicle Registration & RTO Management System

## Overview

A database-driven Vehicle Registration & RTO Management System developed using Python and SQLite to manage vehicle registration records efficiently.

## Features

- Vehicle registration management
- Special number allocation
- Duplicate registration number validation
- Owner record search
- Add, update, delete, and search vehicle records (CRUD operations)
- SQLite database integration
- Input validation and exception handling

## Technologies Used

- Python
- SQLite

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the application:

```bash
python Vehicle_Registration_RTO_Management_System.py
